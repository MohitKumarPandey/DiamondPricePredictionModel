<h1 align="center">💎 Diamond Price Prediction using Machine Learning</h1>

<p align="center">
A Machine Learning Web Application that predicts the price of a diamond based on its physical characteristics using <b>Linear Regression</b>.
</p>

<p align="center">
Python • Scikit-Learn • Streamlit • Pandas • NumPy
</p>

## 📌 About Project

This project predicts the estimated price of a diamond based on the following features:<br><br>

💎 Carat<br>
💎 Cut<br>
💎 Color<br>
💎 Clarity<br>
💎 Depth (%)<br>
💎 Table (%)<br>
💎 Length (x)<br>
💎 Width (y)<br>
💎 Height (z)<br><br>

The model has been trained using the <b>Linear Regression</b> algorithm and deployed using <b>Streamlit</b>.
## 🚀 Features

✔ Modern Streamlit User Interface<br>
✔ Instant Price Prediction<br>
✔ Label Encoding for Categorical Features<br>
✔ Linear Regression Model<br>
✔ Clean & Responsive Design<br>
✔ Fast Prediction Results
## 📊 Dataset Information

**Dataset:** Diamonds Dataset

### Features

| Feature | Description |
|---------|-------------|
| Carat | Weight of Diamond |
| Cut | Quality of Cut |
| Color | Diamond Color Grade |
| Clarity | Diamond Clarity Grade |
| Depth | Total Depth (%) |
| Table | Width of Top Surface (%) |
| x | Length (mm) |
| y | Width (mm) |
| z | Height (mm) |

### Target

**Price (USD)**
